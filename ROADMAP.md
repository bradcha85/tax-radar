## v1.0 — MVP 출시

> "쓸 수 있는 앱"

### 포함 기능

| 기능 | 설명 |
|------|------|
| 부가세 예측 | 반기별 부가세 예상 납부액 |
| 종소세 예측 | 연간 종합소득세 예상 납부액 |
| 매출/경비/의제매입 입력 | 월별 데이터 관리 |
| 세금 상세 분석 | 항목별 분해 + 계산 과정 |
| 시뮬레이터 | what-if 매출 변동 시뮬레이션 |
| 정밀 종소세 | 5단계 위저드 (인적공제, 노란우산 등) |
| 정확도 게이지 | 입력 완성도 시각화 |
| 세금 캘린더 | 부가세/종소세 납부 기한 D-day |
| 용어사전 | 세금 용어 해설 + 즐겨찾기 |
| 온보딩 | 3단계 초기 설정 플로우 |

## 핵심 원칙

1. **예측 도구, 신고 도구 아님** — 삼쩜삼/SSEM과 경쟁하지 않는다
2. **오프라인 우선** — 서버 의존 기능은 선택적으로만 추가
3. **음식점/카페 특화** — 범용화하면 차별점이 사라진다
4. **무료 유지** — 초기 사용자 확보가 과금보다 중요
5. **간단함이 경쟁력** — 입력 부담을 늘리는 기능은 신중하게
