# 용어 설명 UI 카피 (초간단)

목표: “어려운 세금 용어”를 **제목 12자 이하 + 2줄 설명**으로 끝내고, `어디서 찾기`로 바로 안내한다.

## UI 규칙

- **제목(title)**: 12자 이하, 명사형(예: `기납부세액`)
- **2줄 설명(line1/line2)**: 각 줄은 짧게(권장 18~24자 내), 쉬운 말로
- **CTA 버튼 문구**: 기본은 `어디서 찾기` (일관 UX)
- **가이드(선택)**: CTA 탭 시 열리는 “찾는 법” 안내는 3~5줄로 끝내기

## “어디서 찾기” 가이드(요약 텍스트)

아래 문구는 `어디서 찾기` 탭 시 열리는 바텀시트/모달의 기본 본문으로 사용한다.

### T01 종소세
- 홈택스 “종합소득세 신고” → 결과 화면에서 확인
- 종소세는 보통 5월 신고/납부 시즌에 확정돼요

### T02 지방소득세
- 종소세 신고 결과에 함께 표시돼요
- 보통 “지방소득세(10%)”로 따로 보입니다

### T03 총세금(결정)
- 홈택스 신고 마지막 단계 “결정세액”을 확인하세요
- ‘총세금’은 (종소세+지방소득세) 기준으로 안내합니다

### T04 기납부세액
- 근로/기타소득: 원천징수영수증/지급명세서의 “원천징수세액”
- 중간예납: 고지서 또는 홈택스 납부내역

### T05 추가/환급
- “총세금 − 기납부”로 자동 계산돼요
- 값이 +면 추가 납부, −면 환급이에요

### T06 총소득
- 사업/근로/연금/금융/기타 등 각 소득을 합산한 값이에요
- 해당되는 소득만 입력하면 자동으로 더해줘요

### T07 사업소득
- 장부/매출자료(카드, 현금영수증, 세금계산서 등)를 참고하세요
- “매출(VAT 제외) − 필요경비”가 핵심이에요

### T08 필요경비
- 카드/현금영수증, 세금계산서, 거래명세서, 임차료/인건비 내역 등
- ‘부가세용 매입’과 ‘종소세용 경비’는 기준이 달라요(전체 경비가 종소세 기준)

### T09 소득금액
- 홈택스 신고서/신고 도움말에서 “소득금액” 항목을 확인하세요
- 직접 계산하면 “소득 − 필요경비”에 가까워요

### T10 과세표준
- 홈택스 신고서에 “과세표준” 항목으로 표시돼요
- 공제 적용 후 남는 금액이 과세표준입니다

### T11 누진세율
- 과세표준 구간별로 세율이 달라요
- 앱은 구간별 누진 방식으로 계산합니다

### T12 산출세액
- 신고서의 “산출세액” 항목을 확인하세요
- 공제/감면 전의 기본 세금입니다

### T13 소득공제
- 신고서의 “소득공제” 항목을 확인하세요
- 과세표준을 줄여주는 항목입니다

### T14 세액공제
- 신고서의 “세액공제” 항목을 확인하세요
- 계산된 세금에서 직접 빼는 항목입니다

### T15 인적공제
- 가족 구성(본인/배우자/부양가족)을 기준으로 입력합니다
- 홈택스 신고에서 인적공제 입력 단계가 있어요

### T16 노란우산
- 노란우산 납입내역/증명서에서 연 납입액 확인
- 보통 “월 납입액 × 12”로 입력해도 좋아요

### T17 근로소득
- 연말정산 원천징수영수증을 참고하세요
- “근로소득(또는 소득금액)”과 “원천징수세액”이 핵심입니다

### T18 원천징수
- 원천징수영수증/지급명세서의 “원천징수세액”을 확인하세요
- “이미 낸 세금”으로 합산됩니다

### T19 중간예납
- 종합소득세 중간예납 고지서 또는 홈택스 납부내역에서 확인
- 있으면 ‘기납부세액’에 꼭 포함해야 정확해요

### T20 금융 과세
- 금융기관(증권/은행) 연간 명세를 참고하세요
- 이자·배당이 큰 경우 다른 소득과 합산될 수 있어요

### T21 귀속연도
- 귀속연도는 “소득이 발생한 해”예요
- 보통 작년 귀속을 다음 해 5월에 신고/납부해요

### T22 정밀도
- 입력이 많을수록 점수가 올라가요
- `추정`이나 `미입력`이 많으면 낮아져요

### T23 추정
- `모르겠어요`를 누르면 평균/추정치로 계산돼요
- 결과 화면에 `추정 포함` 배지가 떠요

### T24 기장신고
- 홈택스 종합소득세 신고에서 “간편장부/복식부기” 선택을 참고하세요
- 장부로 매출·경비를 정리했다면 기장신고에 가까워요

### T25 단순경비율
- 경비를 정확히 모를 때 “업종별 경비율”로 비용을 추정해요
- 홈택스 신고 도움말/국세청 안내에서 “단순경비율”을 확인할 수 있어요

### T26 분리과세
- 일부 소득(예: 금융소득)은 조건에 따라 “따로 과세”될 수 있어요
- 분리과세면 종소세(합산) 계산에 포함되지 않을 수 있어요

### T27 종합과세
- 다른 소득과 합쳐서(합산) 종소세를 계산해요
- 금융소득이 큰 경우 종합과세 대상이 될 수 있어요

### T28 증빙
- 카드매출/현금영수증/세금계산서, 원천징수영수증 같은 자료예요
- 숫자 입력이 어려우면 “증빙 보고 그대로”가 가장 정확해요

## (추가) 부가세 “어디서 찾기” 가이드(요약)

### V01 부가세
- 홈택스/손택스 “부가가치세 신고/납부” 메뉴에서 확인
- 보통 1기(상반기), 2기(하반기)로 나뉘어요

### V02 납부세액
- 부가세 신고 결과의 “납부할 세액” 또는 “환급세액”을 확인
- 앱은 환급 가능성이 있으면 0원으로 보일 수 있어요(설명 배지 필요)

### V03 매출세액
- 판매(매출)에서 발생한 부가세예요
- 매출이 ‘VAT 포함’이면 총매출에서 자동으로 분리됩니다

### V04 매입세액
- 지출 중 “부가세가 붙는 매입”에서 발생한 부가세예요
- 영수증/세금계산서가 있으면 정확도가 올라가요

### V05 의제매입
- 면세 식재료(농수산물 등) 매입액을 입력하면 공제가 반영돼요
- “전체 지출”이 아니라 “면세 식재료 매입”만 해당돼요

### V06 신카 공제
- 카드/현금영수증 발행 금액에 따라 일부 공제돼요
- 연간 한도에 걸릴 수 있어요(해당 시 안내 필요)

### V07 과세기간
- 1기: 1~6월, 2기: 7~12월 기준으로 입력을 모아 계산해요
- 기간이 바뀌면 “최근 6개월 입력”도 바뀝니다

### V08 VAT 포함
- “입력한 매출이 소비자 가격(부가세 포함)인지” 여부예요
- 모르면 보통 포함이지만, 업종/가격표에 따라 다를 수 있어요
